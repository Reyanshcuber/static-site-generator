**Static Site Generator-**

Use this to convert markdown to html!

Clone the repositiory and follow the steps-

Edit index.md to your liking, you can also add several web pages. After that, run main.py, open the html file and make a local server if you want!

That's it! If you find any bugs or have feedback, send a pull request.



**This project was guided by the static site generator course on boot.dev**
